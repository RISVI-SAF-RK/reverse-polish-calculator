# Reverse Polish Calculator

## Project Description
This project implements a Reverse Polish Calculator using Stack Data Structure in C programming language.

## Features
- Push numbers into stack
- Addition
- Subtraction
- Multiplication
- Division
- Display top value

## Technologies Used
- C Programming
- Stack Data Structure
- Code::Blocks IDE

## Sample Operations

### Addition
3 4 +

### Multiplication
6 7 *
